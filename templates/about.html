About this Project

> So this project is an LMS or learning management system sponsored by Lynchburg Baptist Association

>This project is aimed at advancing the Kingdom of God, through serving others around the world by teaching English while also building disciples of Jesus Christ

>This project is still very much in the development phase as many bugs need to be fixed and much more functionality must be added

>This project is built using HTML, CSS, Bootstrap and Python/Flask on the backend

Run this project

>I highly recommend that you do not run this project locally as it will require you to have a local postgres database setup and configured to this application

>This project will also require you to use Python along with Python packages such as Flask, SqlAlchemy ORM and jinja templating engine
